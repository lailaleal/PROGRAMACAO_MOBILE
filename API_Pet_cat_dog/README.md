# PetPulse - Explorador de Raças

Este é um aplicativo Android desenvolvido em Kotlin utilizando **Jetpack Compose** para a disciplina de Programação Mobile. O app consome a [The Cat API](https://thecatapi.com/) para exibir informações sobre diferentes raças de gatos.

## Funcionalidades
- **Busca Dinâmica:** Pesquise raças pelo nome.
- **Lista de Raças:** Exibição clara com fotos e temperamento.
- **Gestão de Estados:** Tratamento de carregamento (Loading) e erros de conexão.
- **Design System:** Baseado na identidade visual *Kindred Paw*.

## Tecnologias Utilizadas
- **Retrofit & Moshi:** Para requisições HTTP e parsing de JSON.
- **Coil:** Carregamento assíncrono de imagens.
- **Jetpack Compose:** Interface reativa e moderna.
- **ViewModel & StateFlow:** Arquitetura MVVM para gerenciamento de estado.

## Como Executar
1. Clone o repositório.
2. Abra o projeto no **Android Studio (Hedgehog ou superior)**.
3. Sincronize o Gradle.
4. Execute o app em um emulador ou dispositivo físico.
